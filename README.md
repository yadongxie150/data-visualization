# data-visualization

数据可视化的实践



### 目前用到的可视化库主要有一下几种：

- Echarts: 基于百度自己研发的Canvas类库ZRender，上手简单，文档清晰
- ReCharts: 阿里集团数据技术和产品部开发，基于React 和 D3
- Antv: 蚂蚁金服体验技术部可视化方案
- BizCharts: Antv-G2基于React的封装
